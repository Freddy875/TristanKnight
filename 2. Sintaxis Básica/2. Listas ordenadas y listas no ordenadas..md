Las listas en Markdown pueden ser ordenadas o no ordenadas. Para crear una lista no ordenada, puedes utilizar guiones, asteriscos o signos de suma:

```markdown
- Elemento 1
- Elemento 2
- Elemento 3
```

o

```markdown
* Elemento A
* Elemento B
* Elemento C
```

o

```markdown
+ Elemento X
+ Elemento Y
+ Elemento Z
```

Esto se verá como una lista con viñetas:

- Elemento 1
- Elemento 2
- Elemento 3

Para listas ordenadas, simplemente utiliza números seguidos por un punto:

```markdown
1. Primer elemento
2. Segundo elemento
3. Tercer elemento
```

Y se verá así:

1. Primer elemento
2. Segundo elemento
3. Tercer elemento
